QuickTools Privacy Policy

This Privacy Policy applies to the QuickTools mobile application developed by Codex12dev. QuickTools is provided “AS IS”.

QuickTools is a utility app that includes tools such as Password Generator and Backup, QR Scanner and Generator, Tasks, Flashlight, Internet Speed Monitor, Wi-Fi File Server, Tap to Lock, Widgets, and Quick Settings Tiles.

Information Collection and Use

Most QuickTools features can be used without signing in.

QuickTools offers optional Google Sign-In for account features such as Password Backup and Cloud Sync. If you choose to sign in, QuickTools may collect and use the following account information:

  • Name
  • Email address
  • User ID

QuickTools may display your Google account profile photo inside the app if it is provided by Google Sign-In, but QuickTools does not store it separately for app features.

This information is used only for:

  • App functionality
  • Account management
  • Showing your signed-in account inside the app
  • Enabling optional Password Backup and Cloud Sync

QuickTools does not sell user data. QuickTools does not use user data for advertising or marketing. QuickTools does not use third-party analytics or user tracking services in the current version.

Password Generator, Saved Passwords, and Cloud Sync

QuickTools includes a Password Generator and Saved Passwords feature.

Passwords saved locally are stored on the user's device. Where supported by Android, saved passwords are protected using device-level local encryption.

QuickTools also offers optional Cloud Sync for saved passwords. Cloud Sync is not required to use the app. If you enable Cloud Sync, your saved password backup data may be stored in the cloud for backup and restore purposes.

Before cloud upload, password backup content is encrypted by the app. Your Cloud Backup Password is used to protect your cloud backup and is not uploaded by QuickTools.

Cloud Sync may store:

  • Encrypted password backup data
  • Cloud Trash data
  • Account-related cloud sync status
  • Data needed to backup, restore, delete, or sync your saved passwords

QuickTools does not use saved passwords for advertising, analytics, or marketing.

Tasks and Notes

QuickTools may store user-created tasks or notes locally on the user's device.

Tasks and notes are not uploaded to QuickTools cloud services in the current version. They remain on the device unless deleted by the user, app storage is cleared, or the app is uninstalled.

QR Scanner and QR Generator

QuickTools includes QR scanning and QR generation features. QR scanning may use the device camera or images selected by the user from the device gallery.

QR code content is processed locally on the device. QuickTools does not upload scanned QR content or generated QR images to external servers.

Wi-Fi File Server

QuickTools includes an optional Wi-Fi File Server feature that allows users to transfer files between their device and another device on the same local network.

Files transferred through this feature are handled locally over the user's network. QuickTools does not upload these files to external servers.

Internet Speed Monitor

QuickTools includes an optional live internet speed monitoring feature that displays upload and download speed through the app and, when enabled by the user, through a persistent notification.

This feature processes network traffic statistics locally on the device and does not transmit browsing activity or personal network data to QuickTools servers.

Flashlight, Widgets, Tap to Lock, and Quick Settings Tiles

QuickTools includes flashlight, home screen widget, Tap to Lock, and Quick Settings tile features. These features are used only to perform user-requested actions on the device.

The Tap to Lock feature may require Device Administrator permission. This permission is used only to lock the device when the user requests it. QuickTools does not use this permission to access personal content.

Third-Party Services

QuickTools may use services provided by Google, Android, Google Play, Google Sign-In, and cloud service providers for app functionality, account sign-in, security, app distribution, and optional cloud backup features.

These services may process data according to their own privacy policies and terms.

QuickTools does not currently use third-party advertising, analytics, or user tracking services.

Data Sharing

QuickTools does not sell user data.

QuickTools does not share user data with advertisers or marketing companies.

Data may be processed by service providers such as Google-provided services or cloud service providers only as needed to provide app functionality, account sign-in, cloud backup, app security, and app distribution.

Data Retention and Deletion

Local data stored on the user's device remains on the device until the user deletes it, clears app storage, or uninstalls QuickTools.

Cloud account data and cloud password backup data remain until the user deletes their QuickTools account and associated cloud data.

Users can delete their QuickTools account and associated cloud data inside the app:

  1. Open QuickTools.
  2. Go to the Account page.
  3. Tap Delete Account.
  4. Type DELETE to confirm.
  5. Complete Google reauthentication if asked.

If users cannot access the app, they can request account and cloud data deletion using the Delete Account page.

Deletion requests should be sent to: codex12dev@gmail.com

For your security, deletion requests must be sent from the same Google account email used to sign in to QuickTools. Requests from a different email address may not be processed.

Verified deletion requests are processed within 5 to 7 business days.

When a QuickTools account is deleted, the following cloud data linked to that account may be deleted:

  • QuickTools account data
  • Cloud password backup data
  • Cloud Trash data
  • Cloud data linked to the QuickTools account

Local-only data stored only on the user's device cannot be deleted remotely by Codex12dev. To remove local-only data, users can delete it inside the app, clear app storage, or uninstall QuickTools.

Children’s Privacy

QuickTools is not specifically targeted at children under 13.

Codex12dev does not knowingly collect personal information from children under 13.

Security

QuickTools uses reasonable security measures to protect user data.

Data transmitted between the app and cloud services is transferred over secure connections.

Saved passwords are protected locally using device-level encryption where supported by Android.

Cloud password backup content is encrypted before upload when Cloud Sync is enabled.

QuickTools uses reasonable safeguards to help protect user data during storage and transfer.
